# Instagram Clone

A dynamic web app built with **ReactJS**, **Flask**, and custom REST APIs, emulating core Instagram features:

- Photo uploads with client‐side preview  
- Likes, comments, and user follow/unfollow  
- Session management with secure authentication  
- Infinite scroll feed for seamless browsing  

_Originally deployed to AWS for scalability and reliability._
